# 01 — What Is Flutter?

## 📖 The simple definition

**Flutter** is a free toolkit from Google. It lets you build one app that runs on Android, iPhone, web, and desktop — using **one codebase**.

> One codebase. Many platforms. Native speed.

## 🤔 Why does Flutter exist?

Before Flutter, one app meant two projects:
- **Java/Kotlin** for Android
- **Swift** for iPhone

Two codebases. Two sets of bugs. Double the work for every new feature.

Flutter fixes this. Write the app once. Run it everywhere.

## 🧩 Quick facts

| Fact | Detail |
|------|--------|
| Made by | Google |
| First stable release | 2017 |
| Language used | Dart |
| Runs on | Android, iOS, web, Windows, macOS, Linux |
| Famous apps using it | Google Pay, BMW app, eBay Motors |

## 🧠 The one-line idea to remember

> Flutter **draws every pixel of your app itself**. It does not ask Android or iPhone to draw the buttons for you.

This is why a Flutter app looks exactly the same on every device. The next file explains how it actually does this.

## ✅ Key Takeaways

- Flutter = toolkit + Dart language → one app, many platforms
- Made by Google, stable since 2017
- Solves the "build it twice" problem
- It paints its own UI — it doesn't borrow the phone's native buttons
