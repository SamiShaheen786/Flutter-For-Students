# Chapter 00 — Introduction to Flutter

Welcome to **Flutter-For-Students** 🎯

Before you write a single line of widget code, you need three answers:
- What is Flutter?
- What does it actually do?
- What is a widget?

This chapter answers all three.

## 🎓 Who this is for

You should already know:
- Java basics (classes, objects, methods)
- Basic Dart (if you finished Dart-For-Students, you're ready)

No mobile or UI experience needed.

## ✅ By the end of this chapter, you can:

- Explain Flutter in one simple sentence
- Explain why Flutter doesn't use native phone buttons/screens
- Say what a "widget" is
- Read a basic `main.dart` file and understand it
